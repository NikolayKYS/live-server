# live-server

This is a little development server

## Installation

```bash
$ git clone https://github.com/NikolayKYS/live-server.git
$ cd live-server
$ chmod +x install.sh
$ ./install.sh
```

## Usage

```bash
$ live-server [file] [-P port (default :8000)]
```
